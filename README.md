
# 🎅 聖誕老人集合廣場 Santa-2025

## Demo 網址

[https://sodahsu.github.io/santa-2025/](https://sodahsu.github.io/santa-2025/)

## 專案簡介

本專案是一個聖誕派對互動網頁，讓大家可以用手機掃描 QR Code，上傳大頭照和暱稱，並即時出現在大螢幕的聖誕廣場上。現場還有多種互動特效，讓派對更歡樂！

- 支援手機掃描加入，免安裝 App
- 即時同步顯示所有參加者
- 點擊頭像可觸發趣味特效
- 派對模式自動啟動雪花、Disco、互動動畫

## 如何操作

### 1. 大螢幕端（廣場模式）
1. 直接開啟 `index.html`，即可進入聖誕廣場。
2. 螢幕上方會顯示 QR Code，供現場參加者掃描。
3. 廣場會即時顯示所有已加入的聖誕老人頭像。
4. 點擊任一頭像可觸發互動動畫。

### 2. 手機端（上傳模式）
1. 用手機掃描大螢幕上的 QR Code。
2. 進入上傳頁面，輸入暱稱並選擇/拍攝大頭照。
3. 點擊「發送上大螢幕」即可加入廣場。
4. 幾秒後即可在大螢幕看到自己的頭像和暱稱。

### 3. 派對特效
- Disco 派對：點擊「啟動派對」按鈕或等待自動觸發，廣場進入 Disco 模式。
- 雪花特效：自動在大螢幕飄落。
- 隨機互動：頭像會自動觸發「大風吹」、「內向模式」等趣味動畫。


## 主要功能
- 手機掃描 QR Code 即可加入聖誕廣場
- 上傳大頭照、暱稱與自訂狀態，抽取隨機聖誕裝備（扭蛋機制）
- 所有參加者即時同步顯示於大螢幕，支援互動點讚、趣味動畫
- 派對模式自動啟動雪花、Disco、隨機互動（如大風吹、內向模式）
- 點擊頭像可顯示自訂狀態、暱稱，並觸發特效

## 技術架構
- 前端：純 HTML / CSS / JavaScript，無需安裝任何套件
- 後端：Firebase Firestore（雲端即時資料庫，負責儲存與同步所有參加者資料）
- QR Code 產生：api.qrserver.com（動態產生掃描連結）
- 音樂、動畫、特效皆於前端實現
